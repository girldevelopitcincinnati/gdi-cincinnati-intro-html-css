SECTION 1

Inline & Block Elements

Inline vs Block

So far, we have mostly seen "block" elements. They appear on the next line, like paragraphs.

There are also "inline" elements. They appear on the same line that they are written on, like links.

Block & Inline Elements

  • CSS divides elements into two types: inline and block.
  • Browsers render a new line after each block element.
  • Inline elements: span, img, a, br, em, strong
  • Block elements: div, p, h1, ul, li, almost everything else

Element: Div

<div></div>
  • A division, or section of content.
  • Generic block element. Each div renders on a new line.
  • Use it to group HTML elements to format them with CSS. 
    <div>
  <p>Content<p>
  <p>Content<p>
</div>

Grouping elements with div

The div tag can be used to group elements into sections.

Apply IDs and Classes to control the styles with CSS.

<div id="header">
  <h1>Main Heading<h1>
</div>
<div class="sub-content">
  <p>Some more content</p>
</div>

SECTION 2

CSS Pseudo-Classes

Pseudo-classes

Pseudo-classes add special effects to some CSS selectors

  • Appears after the standard selector
  • Begins with a colon :
selector:pseudo-class{
  property:value;
}
For example, you can use pseudo-classes to change the format of a link when you hover over it. 
a:hover{
  text-decoration: none;
}

Link Pseudo-classes

a:link  {color:#FF0000;} /* unvisited link */
a:visited {color:#00FF00;} /* visited link */
a:hover, a:focus {color:#FF00FF;} /* mouse over or select with keyboard*/
a:active {color:#0000FF;} /* selected link */

Link pseudo-classes must be listed in this order.

SECTION 3

The Box Model

Box Model

Padding

Space between the border and the content

Notice that the padding adds to the total width of the box.

Padding

15 pixels on all sides

p { padding: 15px; }

10 pixels on top only

p { padding-top: 10px; }

20 pixels on top/bottom, 40 pixels on left/right

p { padding: 20px 40px; }

10 on top, 5 on right, 3 on bottom, 5 on left

p { padding: 10px 5px 3px 5px; }

Padding Syntax

{ padding: all; }                   /* one value for all sides */
{ padding: top/bottom left/right; } /* two values */
{ padding: top right bottom left; } /* four values */

Example of Padding Declaration with 4 Values:

{ padding: 10px 20px 30px 40px; }

How can we keep the order straight?

The values in a shorthand padding declaration are assigned clockwise from the top.

Alternatively, you can remember to get into TRBL — Top-Right-Bottom-Left.

Border

The edge around the box.

Border

Borders are specified as "thickness, style, color".

A solid red border
{ border: 1px solid #ff0000; }
A thick dotted black top border
{ border-top: 4px dotted #000000; }
Two diï¬€erent border styles
{ border-top: 1px solid #ff0000;
  border-bottom: 4px dotted #000000; }

Border: Other Properties

{ border-width: 10px;
  border-style: dashed;
  border-color: #666666; }

You can specify each property separately, or all three together.

Property: Width

Sets the width of an element.

The width value does not include padding or borders.
These will add to the total width.

div#sidebar{
  width: 31%;
  padding: 1%;
  border: none; /* Total width is 33% */
}

Property: Height

Sets the height of an element.

The height value does not include padding or borders.
These will add to the total height.

p.alert{
  height: 50px;
  padding: 5px;
  border: 2px solid red; /* Total height is 64px */
}

Auto Margin

If a margin is set to auto on a block element that has width, that margin will take up as much space as possible.

Centered
{ width: 300px;
  margin: auto; }
Flush-Right
{ width: 300px;
  margin-left: auto;
  margin-right: 5px; }

Wrappers

Wrappers are a good way to center content if the user's screen width is wider than your content.

div.wrapper {
  width: 100%;
  max-width: 1400px;
  margin: 0 auto;
}
